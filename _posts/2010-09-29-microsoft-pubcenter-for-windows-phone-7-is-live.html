Today is a big day since my team opening Microsoft PubCenter for Windows Phone 7 developers. Starting from today every Windows Phone 7 Developer is able to sign up to Microsoft Pub center and create contextual ad unit for Windows Phone 7.

So in order to monetize your future Windows Phone 7 application you need:

  1. Sign Up to Microsoft PubCenter
  2. Download Microsoft Advertising SDK for Windows Phone 7
  3. Create mobile ad unit using Microsoft Pub center
  4. Insert mobile ad control into your application
  5. Submit application to Windows Phone 7 marketplace

Please use Microsoft Pubcenter Community web site to provide feedback and questions.
