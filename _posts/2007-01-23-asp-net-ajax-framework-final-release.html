Congrats all,

Today Microsoft released final version of Microsoft ASP.NET AJAX framework (former ATLAS).
You can download ASP.NET ajax final release components below:

  • ASP.NET AJAX Essential Components

  • ASP.NET AJAX Control Toolkit

  • ASP.NET AJAX Futures January CTP

With ASP.NET AJAX provides developers building client-based Web experiences with a familiar development process and programming model that they already know from using server-side ASP.NET development. Because ASP.NET AJAX is integrated with ASP.NET, developers have full access to the built-in ASP.NET 2.0 application services and the entire .NET Framework.
