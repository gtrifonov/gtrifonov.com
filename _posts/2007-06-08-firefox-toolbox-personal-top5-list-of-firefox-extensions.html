My Firefox toolbox:

  • Firebug – awesome tool for DOM exploring,debugging,css changing and etc. Every UI dev must Have
  • ColorZilla – color picker
  • Web Developer  – great toolbar with validators,cache control setting, css, resizing and etc.
  • IeTab – allows you to switch to use IE for page rendering
  • BlogThis – allow you to create post using Windows Live Writer
